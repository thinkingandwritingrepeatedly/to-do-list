# 我的想法箱 - 使用文档

## 快速开始

直接双击打开 `index.html` 即可使用，无需安装任何软件或搭建服务器。

所有数据保存在你电脑的浏览器本地存储中，关闭页面后数据不会丢失。

## 功能介绍

### 1. 快速添加想法
- 在顶部输入框中输入想法标题，按 Enter 键或点击「添加」按钮即可快速新增
- 点击右侧的「▼」展开详细选项，可以：
  - 选择或输入新的**分类**（如：工作、学习、生活等）
  - 添加**备注**文字
- 添加后输入框自动清空并保持聚焦，方便连续记录

### 2. 管理想法
每个想法支持以下操作：

| 操作 | 说明 |
|------|------|
| 切换状态 | 点击状态图标（圆形）可在「待办 ⇄ 已完成」之间切换 |
| 完成 | 标记为已完成 |
| 暂缓 | 标记为暂时不想做（已搁置） |
| 恢复待办 | 将已完成或已搁置的想法重新设为待办 |
| 编辑 | 修改标题、分类、状态、备注 |
| 删除 | 删除想法（带二次确认） |

### 3. 查看与检索

**状态筛选**
通过「全部 / 待办 / 已完成 / 已搁置」按钮快速筛选不同状态的想法。

**分类筛选**
点击顶部的分类标签，可以只看某个分类下的所有想法。再次点击取消筛选。

**搜索**
在搜索框输入关键词，实时搜索标题和备注内容。

**排序**
支持按最新优先、最早优先、按标题排序。

### 4. 统计信息
页面顶部实时显示：
- 想法总数
- 各状态数量（待办 / 已完成 / 已搁置）
- 各分类条目数

## 数据存储

- 数据保存在浏览器 **localStorage**（本地存储）中
- 页面底部会显示存储位置提示以及当前数据占用空间
- 清除浏览器数据会导致数据丢失，建议定期导出备份
- 不同浏览器之间的数据不互通

## 导出数据

点击页面底部的「导出数据」按钮，将所有想法导出为一个 JSON 文件下载到本地。

导出的 JSON 文件结构：
```json
{
  "version": 1,
  "exportedAt": "2026-05-29T...",
  "todos": [
    {
      "id": ...,
      "title": "...",
      "notes": "...",
      "category": "...",
      "status": "pending",
      "createdAt": "...",
      "updatedAt": "..."
    }
  ]
}
```

## 注意事项

- 建议使用 Chrome、Edge、Firefox 等现代浏览器打开
- 文件路径中不要包含特殊字符
